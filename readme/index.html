所有辅助使用注意事项

使命召唤OL

开启方式步骤：
1：解压软件到桌面 解压密码123
2：管理员运行AERO
3：管理员运行登录器（蓝色的）
4：点功能注册（左上角）
5：登陆
6：安装到目录（游戏目录在什么地址就安装到什么地址）
7：上游戏大厅 窗口化（不可以无边容易闪退）
8：点开启功能
9：HOME呼出
群文件有详细功能视频 开启教程视频！

注意事项：用完辅助之后关闭游戏 再开启登录器点卸载按钮即可！
每次上辅助之前安装即可！防止被检测切记！

堡垒之夜

开启方式步骤如下：
1：解压软件到桌面 解压密码123
2: 管理员运行AERO
3：管理员运行登录器（蓝色的）
4：点功能注册（左上角）
5：登陆
6：加载驱动开启功能后上游戏（如果个别W10无法加载驱动点击关闭校验即可）
7：上游戏窗口化自动弹出功能表

群文件有详细功能视频 开启教程视频！

注意事项：个别W10无法加载驱动点击关闭校验即可在关于里面左上角
关了重新打开登录器管理员运行如果还是失败 进入BIOS关
闭电脑保护即可！
